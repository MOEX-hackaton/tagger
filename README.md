# README

Данный проект посвящен анализу и классификации текстовых новостей и сообщений с использованием методов машинного обучения. Основная цель проекта - автоматизация процесса категоризации текста и выделения ключевых сущностей для улучшения процессов обработки и анализа

# Подготовка данных
Данные предварительно очищены и нормализованы. Предобработка включает удаление специальных символов, HTML тегов, лемматизацию и удаление стоп-слов. После очистки данные токенизируются для дальнейшего анализа.

# Модель Gradient Boosting Classifier и Random Forest Classifier
Используется для классификации текстов по темам и группам тем. Эта комбинация моделей обеспечивает высокую точность и устойчивость к переобучению.

# Требования
Необходимые библиотеки:
- pandas
- scikit-learn
- matplotlib
- nltk
- tqdm
- sklearn_crfsuite


# Результаты
Результаты обучения моделей на тестовой подвыборке:

**Gradient Boosting Classifier и Random Forest Classifier:**
- Группа тем: 0.8675
- Тема: 0.7891
- Тема: 0.5321
